# HttpLoadTester

Учебное приложение для нагрузочного тестирования HTTP-сервисов с графическим интерфейсом.

---

## Что реализовано

- GUI-приложение на WPF для запуска нагрузочных тестов  
- настройка HTTP-запросов (URL, метод)  
- управление количеством запросов  
- настройка параллельной нагрузки (многопоточность)  
- отправка HTTP-запросов  
- обработка ответов сервера  
- отображение HTTP-статусов  
- базовое логирование результатов

## Архитектурная идея

Приложение реализовано как desktop-инструмент с разделением логики:

- **UI (WPF)** — ввод параметров и отображение результатов  
- **Нагрузочный модуль** — управление потоками и генерация запросов  
- **HTTP-клиент** — выполнение сетевых запросов  
- **Логирование** — фиксация результатов и ошибок  

Основная идея — предоставить простой инструмент для тестирования API без сложной настройки.

---

## Основные возможности

### Настройка теста

- задание URL  
- выбор HTTP-метода  
- настройка количества запросов  
- настройка уровня параллелизма  

### Выполнение нагрузки

- генерация множества HTTP-запросов  
- параллельная отправка  
- обработка ответов  

### Результаты

- отображение кодов ответа (200, 404, 500 и др.)  
- фиксация успешных и неуспешных запросов  
- базовая статистика

## Пример использования

1. Ввести URL (например: http://localhost:5000/api/test)  
2. Выбрать HTTP-метод  
3. Указать:
   - количество запросов  
   - уровень параллельности  
4. Запустить тест  
5. Проанализировать результаты  

---

## Ограничения текущего MVP

- нет графиков и расширенной аналитики  
- отсутствуют сценарии тестирования  
- ограниченное логирование  
- нет экспорта результатов  
- отсутствует поддержка авторизации  
- нет конфигурации через файлы
